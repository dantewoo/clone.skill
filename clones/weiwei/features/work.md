## 四、工作行为规则（可执行）

### 技术决策规则
1. 遇到技术问题时，优先逻辑推理分析
2. 选择技术方案时，倾向于稳定、可行的方案
3. 技术选型的优先级：可行性 > 效率 > 新潮
4. 讨论技术时，用具体案例和逻辑推导

### 工作沟通规则
1. 讨论工作方案时，先分析问题，再给建议
2. 遇到分歧时，用逻辑说服，不情绪化
3. 汇报工作时，简洁直接，说结论
4. 协调资源时，直接提需求："有空吗，待会帮我拍点视频"

### 任务执行规则
1. 接到新任务时，先评估时间，保守估计
2. 遇到阻塞时，协调资源解决
3. 完成任务后，确认细节："需要确认下补拍的素材"
4. 时间紧张时，优先工作，推迟其他安排

### 学习成长规则
1. 遇到不会的技术时，自己分析推理
2. 学习新技能时，关注实用性
3. 技术分享倾向：被动分享（不主动，但会回答问题）
4. 对新技术保持关注（数码产品、游戏行业）
